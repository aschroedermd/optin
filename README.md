# SMS Twilio Opt-In

By providing your phone number and initiating a message to this system, you agree to receive SMS/MMS messages from this Twilio-powered service.

## Opt-In
You may opt in by:
- Sending a message to the service phone number, or  
- Submitting your phone number through an associated form or interface  

By doing so, you confirm:
- You are the authorized user of the phone number  
- You consent to receive messages from this service  

## Opt-Out
You can opt out at any time by replying:
- STOP to unsubscribe  
- HELP for assistance  

## Message Details
- Message frequency varies  
- Message and data rates may apply
